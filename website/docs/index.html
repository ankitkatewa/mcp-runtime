MCP Runtime Docs
Overview API Reference

Gateway, policy, and analytics docs for the runtime control plane.

MCP Runtime now ships a Permit-inspired gateway mode, separate access/session resources, and a bundled mcp-sentinel install path. This docs area is the reference for that API surface.

  • MCPAccessGrant and MCPAgentSession resources.
  • Gateway headers, OIDC/JWT bearer-token validation, trust rules, and audit payload fields.
  • mcp-sentinel API endpoints and setup defaults.

Current scope

This reference covers the new gateway, access/session resources, audit fields, and bundled analytics install path. OIDC/JWT bearer-token validation is supported for mcp-sentinel services, but a full OAuth 2.1 authorization server, consent flow, and Dynamic Client Registration are intentionally out of scope for this first cut.
