Welcome

Welcome to MCP Runtime: self-hosted platform for internal MCP servers.

MCP Runtime combines cluster bootstrap, operator-managed MCP server deployment, separate access and session resources, gateway enforcement, registry workflows, and the bundled mcp-sentinel analytics plane in one Kubernetes-first stack.

Start with runtime docs Open API reference

Why MCP Runtime?

  • One control plane for setup, deployment, grants, sessions, and observation.
  • Per-tool policy and trust checks can happen at the gateway instead of only in app code.
  • Registry, ingress, and rollout workflows are part of the same runtime surface.
  • The bundled sentinel plane gives audit, query, and observability paths without another product hop.

Core benefits

Benefit Description
Unified delivery path Use one CLI and one API surface for cluster setup, image flow, manifest generation, and MCP server rollout.
Real access model Separate grants and sessions keep trust ceilings, consent, expiry, and revocation out of deployment-only YAML.
Gateway-aware governance Evaluate policy at tool-call time with identity headers, trust rules, and allow or deny audit decisions.
Built-in analytics plane Bundle ingest, processor, API, UI, gateway, storage, and observability instead of stitching them on later.

Quick start

These commands are the current short path from repo checkout to a running platform flow.

make deps && make build-runtime

./bin/mcp-runtime setup
./bin/mcp-runtime status

# Optional:
# ./bin/mcp-runtime setup --with-tls
# ./bin/mcp-runtime setup --without-sentinel

./bin/mcp-runtime registry push --image my-server:latest
./bin/mcp-runtime pipeline generate --dir .mcp --output manifests/
./bin/mcp-runtime pipeline deploy --dir manifests/

Current scope

Alpha, but broad enough to evaluate

Deployment, routing, grants, sessions, gateway policy, and audit flow are implemented in the current repo. Sentinel supports bearer-token validation, but a full OAuth 2.1 authorization server and Dynamic Client Registration flow are still out of scope.
